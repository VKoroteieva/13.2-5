# Практична робота 13.2: Real-time Stream Processing (Варіант 5 - IIoT Predictive Maintenance)

## Опис архітектури

Проєкт демонструє обробку промислових IoT потоків у реальному часі за допомогою Complex Event Processing (CEP) для виявлення деградації обладнання.

## Реалізований функціонал:

1. **Real-time Event Streams (`items/`)**: Потоки телеметрії (вібрація, температура, тиск станка).
2. **Complex Event Processing (`rules/`)**:
   - Виявлення патернів поломок шляхом кореляції високої вібрації та перегріву.
   - Віконна агрегація (Windowing) для розрахунку ковзного середнього вібрації.
3. **Condition-Based Maintenance**: Генерація тривог при перевищенні порогів надійності.
